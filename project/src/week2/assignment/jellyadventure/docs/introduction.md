# 젤리 어드벤처 게임.

카카오 테크 부트캠프 2기 풀스택 과정 2주차 과제 (상속을 이용한 CLI 어플리케이션 제작)
-> 1주차 과제에 쓰레드를 이용한 기능을 추가합니다.

## 기능

###  시간 체크 기능 추가
  1. 사용자는 자신의 현재 시간을 10초 마다 알 수 있습니다.
  2. 게임 진행 시간을 체크할 수 있습니다.
